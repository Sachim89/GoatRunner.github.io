About

Passion

GoatRunner is a Shuttle service web application to manage university provided shuttle services. This application would enable the students to book cab services at any time of the day.

Responsive

GoatRunner's User Interface contains flexible layouts, flexible images and cascading style sheets. This responsive design builds web pages that detect the visitor's screen size and orientation and change the layout accordingly.

Design

GoatRunner's webpage design is designed to be appealing and user friendly and uses interactive features that deliver pages to your site visitors.
